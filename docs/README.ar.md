# إضافة الإشعارات الفورية لـ osTicket

إشعارات Web Push (PWA) للوحة تحكم الموظفين في osTicket. تُرسل إشعارات فورية عبر المتصفح لأحداث التذاكر، بشكل مستقل تمامًا عن تنبيهات البريد الإلكتروني.

## المميزات
- **إشعارات فورية** لـ: التذاكر الجديدة، الرسائل/الردود الجديدة، التعيينات، التحويلات، التذاكر المتأخرة
- **مستقلة عن تنبيهات البريد الإلكتروني** — تعمل حتى عند تعطيل جميع تنبيهات البريد الإلكتروني
- **تفضيلات الوكيل** مع تبديلات لكل حدث، وتصفية حسب القسم، وساعات هدوء مخصصة
- **ضوابط المسؤول** مع مفتاح رئيسي، وتبديلات لكل حدث، وأيقونة إشعار مخصصة، وإدارة مفاتيح VAPID
- **دعم متعدد اللغات** باستخدام نظام الترجمة المدمج في osTicket
- **متوافق مع الأجهزة المحمولة** مع أيقونتي الجرس والترس في شريط التنقل للجوال
- **متوافق مع الوضع الداكن** (ثيم osTicketAwesome)
- **مبني على Service Worker** — يعمل حتى عند إغلاق تبويب المتصفح
- **بدون تبعيات** — تنفيذ PHP بحت لـ Web Push، لا يتطلب Composer

## المتطلبات
- osTicket **1.18+**
- PHP **8.0+** مع إضافة `openssl` (لتوليد مفاتيح VAPID وتشفير الحمولة)
- HTTPS (مطلوب من قِبل Web Push API)

## التثبيت
1. انسخ مجلد `push-notifications/` إلى `include/plugins/`
2. في لوحة تحكم المسؤول، انتقل إلى **إدارة > الإضافات > إضافة إضافة جديدة**
3. انقر على **تثبيت** بجانب "Push Notifications"
4. اضبط الحالة على **نشط** واحفظ
5. انتقل إلى تبويب **النسخ**، ثم انقر على **إضافة نسخة جديدة**
6. أدخل اسم النسخة واضبط الحالة على **مفعّل**
7. في تبويب **الإعداد**:
   - أدخل موضوع VAPID (مثلاً: `mailto:admin@example.com`)
   - ضع علامة على **تفعيل الإشعارات الفورية**
   - فعّل أنواع التنبيهات المطلوبة
   - اختياريًا أدخل رابط أيقونة الإشعار المخصصة
   - احفظ — سيتم توليد مفاتيح VAPID تلقائيًا

## آلية العمل
### إعداد المسؤول (لوحة تحكم المسؤول > الإضافات > Push Notifications)
| الإعداد | الوصف |
|---|---|
| تفعيل الإشعارات الفورية | مفتاح التشغيل/الإيقاف الرئيسي |
| موضوع VAPID | بريد إلكتروني للتواصل لتعريف خدمة الإرسال |
| مفاتيح VAPID | تُولَّد تلقائيًا عند الحفظ الأول |
| تنبيهات التذكرة الجديدة / الرسائل / التعيين / التحويل / التأخير | تبديلات عامة لكل حدث |
| رابط أيقونة الإشعار | أيقونة/شعار مخصص للإشعارات الفورية (اتركه فارغًا للافتراضي) |

### تفضيلات الوكيل (أيقونة الترس بجانب الجرس في شريط التنقل)
يمكن لكل وكيل تخصيص تفضيلات الإشعارات الخاصة به:
| الإعداد | الوصف |
|---|---|
| تبديلات الأحداث | اختر أنواع الأحداث التي تُشغّل الإشعارات الفورية |
| تصفية الأقسام | استقبل الإشعارات من الأقسام المحددة فقط |
| ساعات الهدوء | تعطيل الإشعارات خلال نطاق زمني محدد (يدعم الفترات التي تمتد إلى اليوم التالي) |

### تدفق الإشعارات
```
هل المفتاح الرئيسي للإضافة مفعّل؟
  └─ هل تبديل حدث الإضافة مفعّل؟ (مثلاً: alert_new_ticket)
      └─ هل للوكيل اشتراك إشعارات فورية؟
          └─ هل تفضيل الوكيل للحدث مفعّل؟
              └─ هل قسم التذكرة ضمن فلتر أقسام الوكيل؟ (فارغ = جميع الأقسام)
                  └─ هل الوقت خارج ساعات الهدوء للوكيل؟
                      └─ إرسال الإشعار ✓
```
> **ملاحظة:** الإشعارات الفورية مستقلة تمامًا عن إعدادات تنبيهات البريد الإلكتروني في osTicket. يمكنك تعطيل جميع تنبيهات البريد الإلكتروني وستستمر الإشعارات الفورية في العمل.

## البنية المعمارية
| الملف | الغرض |
|---|---|
| `plugin.php` | بيان الإضافة (المعرّف، الإصدار، الاسم) |
| `config.php` | حقول إعداد المسؤول + توليد مفاتيح VAPID + إنشاء جداول قاعدة البيانات |
| `class.PushNotificationsPlugin.php` | التهيئة، خطافات الإشارات، مسارات AJAX، حقن الأصول |
| `class.PushNotificationsAjax.php` | وحدة تحكم AJAX (الاشتراك، إلغاء الاشتراك، التفضيلات، الاختبار) |
| `class.PushDispatcher.php` | إرسال الإشعارات مع منطق تحديد المستلمين وتصفية التفضيلات |
| `class.WebPush.php` | مُرسِل Web Push بـ PHP بحت (VAPID + ECDH + AES-128-GCM، بدون Composer) |
| `assets/push-notifications.js` | واجهة الجرس/الترس من جانب العميل، نافذة التفضيلات، تسجيل Service Worker |
| `assets/push-notifications.css` | تنسيق أيقونات التنقل، النافذة المنبثقة، مفاتيح التبديل، الوضع الداكن |
| `assets/sw.js` | Service Worker لاستقبال الإشعارات الفورية وعرضها |

## جداول قاعدة البيانات
تُنشئ الإضافة جدولين عند الحفظ الأول في الإعداد:
- `ost_push_subscription` — يخزن نقاط نهاية اشتراك الإشعارات الفورية للمتصفح لكل وكيل
- `ost_push_preferences` — يخزن تفضيلات الإشعارات لكل وكيل
